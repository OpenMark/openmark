Basic system status

Memory used: %%MEMORY%%
Total memory in use by the Tomcat instance, not just memory for Om.
Sessions: %%SESSIONS%%
User sessions in memory. Sessions expire after 8 hours of disuse.
Temp forbids: %%TEMPFORBIDS%%
When users are temporarily forbidden to log on here. This happens when they connect to another server, in order to prevent timing issues. Forbids only persist for 1 minute.
DB connections: %%DBCONNECTIONS%%
The system maintains multiple database connections for use by different threads. There is a maximum of 5 connections. After a connection has been out of use for 15 minutes, it is closed.
