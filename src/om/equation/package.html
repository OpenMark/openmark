Code for converting a small subset of LaTeX format to XML and then for displaying equations in the XML format (either as bitmaps or converting to XML). This code provides the underlying behaviour of the <equation> and <eq> question components, but could be used in other situations too.

Begin with static methods in the Equation class for most uses.

The '.generated' package is created by the Ant build script using JavaCC and JJTree utilities, based on the grammar specification grammar.jjt. Don't mess with anything in there.
